Image Gallery Setup Instructions

1. Create a folder named 'images' in the project directory.
2. Add at least 5 images to this folder and name them:
   - islambad.jpg
   - islambad1.jpg
   - islambad2.jpg
   - islambad3.jpg
   - islambad4.jpg
3. Open index.html in your browser to view the image gallery.

You can use your own images or download sample images from the internet.
